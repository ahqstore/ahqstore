<TabItem label="English">

The configuration has been updated to add support for 
- Native Windows Arm64 binaries (not the x86 version running under prism)
- Linux arm64 support
- Linux arm32 support

Also, the team is working on an Android Build targetting Android 11 or above

</TabItem>

<TabItem label="Español">

La configuración se ha actualizado para agregar compatibilidad con:
- Binarios nativos de Windows Arm64 (no la versión x86 que se ejecuta en Prism)
- Compatibilidad con Linux arm64
- Compatibilidad con Linux arm32

Además, el equipo está trabajando en una compilación de Android destinada a Android 11 o superior

</TabItem>

<TabItem label="हिंदी">
कॉन्फ़िगरेशन को अपडेट करके इसके लिए समर्थन जोड़ा गया है
- नेटिव विंडोज आर्म64 बाइनरी (प्रिज्म के तहत चलने वाला x86 संस्करण नहीं)
- लिनक्स आर्म64 समर्थन
- लिनक्स आर्म32 समर्थन

साथ ही, टीम एंड्रॉइड 11 या उससे ऊपर के वर्जन को लक्षित करने वाले एंड्रॉइड बिल्ड पर काम कर रही है
</TabItem>
