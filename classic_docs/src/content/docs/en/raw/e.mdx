<Aside type="caution" title="Beware">
We recommend this only when there's no prebuilt binaries available. It'll also try to run the official installer
</Aside>

<Steps>
  1. Clone the [GitHub repo](https://github.com/ahqsoftwares/tauri-ahq-store)
  2. Select the OS you're building it for
</Steps>

<Tabs>
  <TabItem label="Windows">
    <Steps>
      3. Install Rust
      4. Install NodeJS
    </Steps>
    ```sh
    npm i -g pnpm
    rustup install nightly
    pnpm install
    pnpm tauri build
    cd src-service
    cargo build --release
    cd ..
    ```
    Follow step 3 of the windows tutorial
  </TabItem>
  <TabItem label="Linux">
  ```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustup install nightly
sudo apt update
sudo apt install libwebkit2gtk-4.1-dev build-essential curl wget file libssl-dev libayatana-appindicator3-dev librsvg2-dev
curl -fsSL https://get.pnpm.io/install.sh | sh -
pnpm install
pnpm tauri build
cd src-service
cargo build --release
cd ..
```
Follow step 3 of the linux tutorial
  </TabItem>
</Tabs>
