<TabItem label="English">
As you would already know that AHQ Store didn't really supported Windows 7 & 8.1 that well, we've decided to drop support of it altogether.

There's really not much to say here either. Still following are some reasons:

### Microsoft Edge and WebView2 ending support for Windows 7 and Windows 8/8.1

<Aside title="Source">
  [Microsoft](https://blogs.windows.com/msedgedev/2022/12/09/microsoft-edge-and-webview2-ending-support-for-windows-7-and-windows-8-8-1)
</Aside>
> - To get new versions of Microsoft Edge or the WebView2 Runtime with up-to-date
functionality and security, please upgrade to Windows 10 or later. > - We also encourage
developers to end support for Windows 7 and Windows 8/8.1. We acknowledge that this
may not be easy for some developers to do, however ending support for these operating
systems will help keep end users safe from potential security threats and risks as
both operating systems go out of support on January 10th, 2023.

### Windows support schedule 2024 (Rust Lang Compiler Team)

<Aside title="Source">
  [Rust Lang - Compiler Team - Issue 651 -
  GitHub](https://github.com/rust-lang/compiler-team/issues/651)
</Aside>
> - Currently Windows 7 and 8 are listed as Tier 1 supported platforms. However,
this has not been true for a long long time. We simply do not have the testing infrastructure.
And with so much software now abandoning Windows 7 and 8 (everything from Git to
Go to every major browser), continuing to provide the little support that we do is
only getting more difficult. There is also a lack of vendor support for these targets.
> - This proposal aims to raise the minimum supported Windows version of current
targets to Windows 10 (first released in 2015) during 2024. But it also leaves open
the door for new targets that can commit to broader support, By having a separate
target we can ensure there are people specifically tasked with supporting legacy
versions of Windows. > > **Proposal** > - Rust 1.75 will be the last to officially
support Windows 7, 8 and 8.1. Support for Windows versions before 10 will end in
February 2024 with Rust 1.76. > - Support for Windows 7 and 8 may continue beyond
these dates through the creation of new "legacy" targets for older Windows versions.
Individuals or organisation(s) who can commit to providing some level of legacy testing
and support should go through the normal process for creating new targets.

</TabItem>

<TabItem label="Español">
Actualmente, AHQ Store no ha tenido un soporte real para Windows 7 y 8.1. Por lo tanto, hemos decidido abandonar el soporte de estas versiones del sistema operativo.

No hay mucho que decir sobre esto. Sin embargo, aquí hay algunas razones:

### Fin del soporte de Microsoft Edge y WebView2 para Windows 7 y Windows 8/8.1

<Aside title="Fuente">
  [Microsoft](https://blogs.windows.com/msedgedev/2022/12/09/microsoft-edge-and-webview2-ending-support-for-windows-7-and-windows-8-8-1)
</Aside>
> - Para obtener nuevas versiones de Microsoft Edge o el tiempo de ejecución de WebView2
con funcionalidad y seguridad actualizadas, por favor actualice a Windows 10 o una
versión posterior. > - También nosotros recomendamos a los desarrolladores terminar
el soporte para Windows 7 y Windows 8/8.1. Acknowledge que esto puede ser difícil
para algunos desarrolladores, pero al finalizar el soporte para estas versiones del
sistema operativo ayudará a proteger a los usuarios finales de posibles amenazas
y riesgos, ya que ambas versiones del sistema operativo están en desuso desde el
10 de enero de 2023.

### Programa de soporte para Windows 2024 (Equipo del Compilador de Rust Lang)

<Aside title="Fuente">
  [Rust Lang - Compiler Team - Issue 651 -
  GitHub](https://github.com/rust-lang/compiler-team/issues/651)
</Aside>
> - Actualmente, Windows 7 y 8 se listan como plataformas de soporte nivel 1. Sin
embargo, esto no ha sido verdadero durante mucho tiempo. Simplemente no tenemos la
infraestructura de pruebas. Además, con que tanto software ha dejado de soportar
Windows 7 y 8 (desde Git hasta Go hasta todos los navegadores principales), seguir
proporcionando poco soporte es cada vez más difícil. También hay una falta de soporte
de los fabricantes para estas plataformas. > - Este programa busca elevar el nivel
de soporte mínimo de las versiones de Windows actuales a Windows 10 (lanzado en 2015)
en 2024. Sin embargo, deja abierta la posibilidad de que se creen nuevos objetivos
de soporte para versiones anteriores de Windows, a través de la creación de nuevas
"plataformas de soporte legado" para versiones anteriores de Windows. Individuos
o organizaciones que puedan comprometerse a proporcionar algún nivel de soporte de
pruebas y de soporte deberán seguir el proceso normal para crear nuevas plataformas.
> > **Propósito** > - Rust 1.75 será la última versión oficialmente compatible con
Windows 7, 8 y 8.1. El soporte para versiones de Windows anteriores a 10 finalizará
en febrero de 2024 con Rust 1.76. > - El soporte para Windows 7 y 8 puede continuar
más allá de esas fechas a través de la creación de nuevas "plataformas de soporte
legado" para versiones anteriores de Windows. Las personas o organizaciones que puedan
comprometerse a proporcionar algún nivel de soporte de pruebas y de soporte deberán
seguir el proceso normal para crear nuevas plataformas.

</TabItem>

<TabItem label="हिंदी">
आपको जानने की आवश्यकता हो सकती है कि AHQ Store को Windows 7 और 8.1 के साथ को उच्च समर्थन करने में संकेता भूमिका नहीं है। इसलिए हम इसके समर्थन को अलग करना करते हैं।

यह एक काफी कम का समय को नहीं हैं। यहाँ नीचे कुछ कारण हैं:

### Microsoft Edge और WebView2 के लिए Windows 7 और Windows 8/8.1 का समर्थन को अक्षम करने की समय की

<Aside title="स्रोत">
  [Microsoft](https://blogs.windows.com/msedgedev/2022/12/09/microsoft-edge-and-webview2-ending-support-for-windows-7-and-windows-8-8-1)
</Aside>
> - उसले नई संशोदन के लिए Microsoft Edge या WebView2 की संशोदन की संशोदनित और सुरक्षा
के लिए, कृपया Windows 10 या उपस्थित अधिकतर पर अपग्रेड करें। > - हम भी डिव्हलेर्स
को Windows 7 और Windows 8/8.1 के समर्थन को अक्षम करने की सूचना देते हैं। हम समझते
हैं कि यह कहीं सर्दारी से कठिन हो सकता है लेकिन यह समर्थन को संशोदन करने के लिए इस
का अपवाह होने से कोई उपयोगकर्ता या संगठन को सुरक्षित करने के लिए महत्वपूर्ण होता
है, क्योंकि इस दोनों की प्लेटफॉर्म को समाप्ति करने के लिए 10 जनवरी 2023 को पहले हैं।

### Windows के समर्थन का समय कार्यक्रम 2024 (Rust Lang Compiler Team)

<Aside title="स्रोत">
  [Rust Lang - Compiler Team - Issue 651 -
  GitHub](https://github.com/rust-lang/compiler-team/issues/651)
</Aside>
> - सभी तारीख पर Windows 7 और 8 के साथ समर्थित किया जा रहा हैं कि उनका कोशिशात्मक
समर्थन हो। क्याकिं हम समझते हैं कि यह कहीं सर्दारी से कठिन हो सकता है लेकिन हम कभी
भी टेस्टिंग इंफ्रास्ट्रक्चर को नहीं हैं। और सबसे अधिक सॉफ्टवेयर को Windows 7 और 8
के लिए समर्थन को अक्षम करने की अनुमति नहीं देता है (Git, Go और सभी मुख्य ब्राउज़र
को समाप्ति करने के लिए) क्योंकि यह कठिन हो सकता है समर्थन को देखना। यहाँ भी कोई वेंडर
की समर्थन की अनुमति नहीं है। > - यह योजना को समर्थन के लिए करने के लिए किया गया है
कि वर्तमान के तारीख के समर्थित टार्गेट्स को समाप्ति के समय 2024 को Windows 10 (2015
में प्रारंभ किया गया) को समर्थन को उल्लेख करने का प्रयास करें। यह कभी भी नए समर्थित
टार्गेट्स को बनाने के लिए कोशिशा करता है, जो समर्थित करने के लिए कोशिशा करते हैं,
और उन्हें संशोदनित समर्थन को बनाने के लिए कोशिशा करते हैं। समर्थित करने के लिए सही
कोशिशा करने वाले व्यक्तियों या संगठन को समर्थन को बनाने के लिए कोशिशा करने की अनुमति
होती है। > > **योजना** > - रस्ट 1.75 आधिकारिक तौर पर विंडोज 7, 8 और 8.1 के साथ संगत
अंतिम संस्करण होगा। 10 से पहले के विंडोज़ संस्करणों के लिए समर्थन रस्ट 1.76 के साथ
फरवरी 2024 में समाप्त हो जाएगा। > - विंडोज़ के पुराने संस्करणों के लिए नए "विरासत
समर्थन प्लेटफ़ॉर्म" के निर्माण के माध्यम से विंडोज़ 7 और 8 के लिए समर्थन उन तिथियों
के बाद भी जारी रह सकता है। जो व्यक्ति या संगठन कुछ स्तर पर परीक्षण और सहायता प्रदान
करने के लिए प्रतिबद्ध हैं, उन्हें नए प्लेटफ़ॉर्म बनाने के लिए सामान्य प्रक्रिया का
पालन करना चाहिए।
