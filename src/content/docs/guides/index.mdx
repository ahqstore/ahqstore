### Supported Windows Editions
:::caution
  It has been identified that sometimes AHQ Store doesn't work on some specific versions of Windows 10
:::

AHQ Store officially supports the following versions of Windows:
<Tabs>
  <TabItem label="Windows 11 (Recommended)">
    Supported Versions:
      - Windows 11 **21H2**
      - Windows 11 **22H2**
      - Windows 11 **23H2**
      - Windows 11 **24H2**
  </TabItem>

  <TabItem label="Windows 10">
    Supported Versions:
      - Windows 10 **20H2**
      - Windows 10 **21H2**
      - Windows 10 **22H2**
  </TabItem>
</Tabs>


### Stable or Alpha?

<Tabs>
  <TabItem label="Stable (Recommended)">
    The stable version is the tested and working version of AHQ Store.
  </TabItem>

  <TabItem label="Alpha">
    The Alpha version contains pre-released features and may contain bugs. It is meant for devs and highly technical users.
  </TabItem>
</Tabs>
