<Aside type="caution" title="Beware">
We recommend that you use the provided installer to install AHQ Store. This method too will ultimately run the `AHQ Store Installer` when an update will be available
</Aside>

<Steps>
  1. Download the installer and `ahqstore_service` from [here](/download)
  2. Install using the installer file
  3. Now, open an elevated terminal in the folder where `ahqstore_service` is
  . <Code lang="sh" code={'sudo chmod 777 ./ahqstore_service\nahqstore_service'} />
  4. Now, launch the app
</Steps>
